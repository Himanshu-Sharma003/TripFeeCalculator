1. Execute "run.bat" file to compile and execute TripCalculator.

2."TripCalculator.java" is initiator class containing "main"  method. Execute this class to start application.

3."config.properties" and "logging.properties" contains configuration and log properties.

4. Execute "TestExecutor.java" to invoke test case.

5. tripCharges.csv contains fee charged for traveling between point a to point b.

6. Default taps.csv file location is /src/main/java/resources/input.

7. Output file i.e trips.csv is created at root level.


Assumptions:
1. For incomplete Trips, ends date is updated as start date.
